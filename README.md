# Grindstone

A full-stack competitive programming tracker built with Java, 
Spring Boot, PostgreSQL, and React.

## Features
- Log and filter problems by topic, approach, and difficulty
- GitHub-style activity heatmap of your daily practice
- Spaced repetition revision queue for problems to revisit
- Contest log with cumulative rating chart
- Dashboard with streak, stats, and topic breakdown

## Tech Stack
**Backend:** Java, Spring Boot, PostgreSQL, Spring Data JPA  
**Frontend:** React, Recharts, Axios, Vite

## Running locally
Backend runs on `localhost:8080`  
Frontend runs on `localhost:5173`
